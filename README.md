# Neville Goddard - Wiedza i Praktyka

Aplikacja wspierająca naukę i praktykę Prawa Założenia Neville'a Goddarda.

## Funkcje
- **Mentor AI**: Rozmawiaj z mentorem zainspirowanym naukami Neville'a.
- **PWA**: Zainstaluj aplikację na telefonie (iOS/Android) dla szybkiego dostępu.
- **Prywatność**: Klucz API Gemini oraz historia rozmów są przechowywane wyłącznie w pamięci lokalnej Twojego urządzenia.

## Konfiguracja API
Aplikacja korzysta z modelu **Gemini 2.5 Flash Lite**. Aby korzystać z Mentora AI:
1. Wejdź w **Ustawienia**.
2. Wklej swój klucz API Gemini (pobierz go z [Google AI Studio](https://aistudio.google.com/)).
3. Zapisz ustawienia.

## Uruchamianie lokalne
1. `npm install`
2. `npm run dev`

## Instalacja na telefonie (PWA)
1. Otwórz stronę w przeglądarce na telefonie.
2. **iOS (Safari)**: Kliknij "Udostępnij" -> "Dodaj do ekranu początkowego".
3. **Android (Chrome)**: Kliknij menu (trzy kropki) -> "Zainstaluj aplikację".
